# 🚀 Kotlix — Learn Kotlin Visually

Kotlix is an interactive Kotlin learning app designed to help developers understand concepts through **visualization, live demos, and step-by-step explanations**.

Instead of reading documentation, you *see Kotlin in action*.

---

## ✨ Features

### 📦 Modular Learning

* Learn Kotlin concepts as independent modules
* Current module:

  * 🌊 **Flow** — Streams, operators, and behavior

---

### 🔍 Flow Playground

* Explore operators like:

  * `map`, `filter`, `flatMapLatest`, `combine`, `zip`, etc.
* Understand behavior with:

  * Step-by-step explanations
  * Marble diagrams
  * Code snippets

---

### ⚡ Live Demo Panel

* Run real Flow operations
* Provide custom input
* See output instantly

---

### 🔊 Voice Explanation

* Tap steps to hear explanations
* Helps reinforce learning

---
